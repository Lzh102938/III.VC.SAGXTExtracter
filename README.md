# 几乎可用于全版本GTA的GXT查看器

现已支持：
  - GTA III
  - GTA Vice City
  - GTA San Andreas
  - GTA IV

**亦可使用网页端 👉 [https://gxtviewer.pages.dev/](https://gxtviewer.pages.dev/)**（注意：网页端功能可能不全面）

> 同样优秀的GXT工具：[https://github.com/QCJW/GTA-Text-Editing-Tool](https://github.com/QCJW/GTA-Text-Editing-Tool)  
此工具支持GTA5，对GTA4的支持更专业，获得[贴吧汉化组](https://github.com/GTAIV-Complete-Edition-text)官方认可！

---

> 本工具以 Silent 的源代码为基础，加入了可视化界面以及添加了其他 GTA 版本的支持。  
> 原开源链接 👉 [https://github.com/CookiePLMonster/GXT-Extractor](https://github.com/CookiePLMonster/GXT-Extractor)

> 对于 GTA3 版本的 GXT 文件，由于没有表，故程序不会输出 GTA3 的分文本，仅输出集成文本。  
> GTAVC、SA、IV 版本的 GXT 皆会输出集成文本和分文本。  

> 工具内集成了 **码表转换功能**，方便解析 GXT 时遇到二次转换的需求。  

---
## 功能介绍
 - **码表转换**  
    ```
    很多汉化补丁的gxt经过了加密，直接解析看到的是乱码。实际上是一个加密文本，可以通过码表转换工具进行转换。  

    点击 码表转换 功能，可供选择在线文档和本地文档选择，将乱码恢复为正常文本。  
    ```

- **智能翻译**
  ```
  想汉化模组却没有能力翻译文本？别担心，接入的Deepseek是你的左右手！  

  点击 智能翻译 功能，输入你的Deepseek API Key，即可进行翻译。  
  需要注意的是，最后一部分的翻译可能有些慢，请耐心等待。
  ```

- **生成贴图**  
  ```
  电脑无PS？我们能生成贴图！  

  在主界面按下Ctrl+M，打开菜单，选择字库贴图选项，配置参数即可生成贴图。顺带生成字库dat文件！
